Daily pickup limit with live progress — user's count tracked against their personal goal throughout the day
Streak tracker — consecutive days the user meets their goal, displayed with visual continuity on the home screen
Tracking the why — an optional multiple choice prompt that appears after each logged pickup. Options: Boredom, Habit, Social, Work, Anxiety, Other. User can toggle this on or off in settings
Hourly heatmap — visual grid showing which hours of the day have the highest pickup counts
Smart notifications — contextual alerts when the user is approaching or has exceeded their daily limit, plus streak-at-risk reminders
Focus blocks — user defines phone-free time windows and the app tracks whether they stuck to them
